# 開発サーバーの起動
```
npm run dev
```

# 使用技術

## フロントエンド
- 言語：TypeScript
- フレームワーク：Next,js 15
- スタイル：Mantine,css Module
- データfetch：SWR

## バックエンド
- 言語：TypeScript
- フレームワーク：Next.js 15(API Route)
- DB：supabase(postgreSQL)
- ORM：Prisma
- 認証：Supabase Auth
